Source code
~~~~~~~~~~~

The project is hosted on `GitHub <https://github.com/dicomgrid/sdk-python>`_.

Please feel free to create issues in the `bug tracker <https://github.com/dicomgrid/sdk-python/issues>`_ if you find a bug or have some suggestions in order to improve the library.
 

Dependencies
~~~~~~~~~~~~

- Python 3.6+
- *requests*
- *aiohttp*
- *python-box*
- *pydicom*

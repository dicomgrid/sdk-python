Ambra-sdk - An easy way to use AmbraHealth using Python¶

Release v3.21.6.0-1. Installation.

Key features¶

  • Service API

  • Storage API

  • Models for easier sdk usage

  • Some useful addon functions

  • Async

Source code¶

The project is hosted on GitHub.

Please feel free to create issues in the bug tracker if you find a bug or have some suggestions in order to improve the library.

Dependencies¶

  • Python 3.6+

  • requests

  • aiohttp

  • python-box

  • pydicom
